# 🌍 Azure Translator Web App

A modern web-based language translator built using **Azure Cognitive Services**.  
It enables real-time multilingual translation with a clean and responsive interface.

---

## ✨ Features

- 🌐 Real-time text translation  
- 🔤 Supports multiple languages  
- ⚡ Fast and lightweight frontend  
- 🎯 Simple and user-friendly UI  
- ☁️ Powered by Azure Translator API  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Azure Cognitive Services (Translator API)  

---

## ⚙️ How It Works

1. User enters text in the input box  
2. The app sends a request to Azure Translator API  
3. API processes and returns translated text  
4. Output is displayed instantly on the screen
